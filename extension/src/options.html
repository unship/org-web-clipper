org-clipper

Configure how clipped pages are handed to Emacs.

Where each clip is filed is configured in Emacs (org-clipper-target-file / org-clipper-monthly-dir, under org-clipper-target-headline). See the project README.
Capture

Merged with any tags entered in the popup. Allowed characters: A-Z a-z 0-9 _ @ #. Tags are appended to the title as Org's :tag: suffix so the template's %:description picks them up automatically.

Single-letter key of the org-capture-templates entry that handles clips. Default w.

How clips reach Emacs. org-protocol (default) opens an org-protocol://org-clipper?… URL — zero extra process. HTTP POSTs to a local Emacs endpoint (no URL-length limit; best for very long pages). Must match org-clipper-transport in your Emacs config.

Used only when Transport = HTTP. Must match org-clipper-http-port in Emacs.
